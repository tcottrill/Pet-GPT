Commodore PET ROM images go in this folder, one subfolder per ROM set:

    roms\basic2\   BASIC 2, "N" editor
    roms\basic4\   BASIC 4, 40-column
    roms\8032\     BASIC 4, 80-column / 8032 business set

Easiest way: run download-roms.ps1 (next to PetEmu.exe) and it fetches
everything and puts it in the right place:

    powershell -ExecutionPolicy Bypass -File download-roms.ps1

Or by hand: download the plain .bin files below from zimmers.net (no
extraction needed) and drop each into the matching subfolder above.
https://www.zimmers.net/anonftp/pub/cbm/firmware/computers/pet/
